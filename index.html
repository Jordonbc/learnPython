Lesson 0: Introduction

Why might you want to learn Python?

Well Python has a wide range of applications ranging from science, games and just about anything and it is the easiest computer language to learn.

How much will it take to learn Python?

Well, Python is what is known as a very high level programming language. This means that Python is more towards human language rather than machine language.

Lesson 2: Hello World!

The most simple program to write is the Hello World! program. This program will just output Hello World! To the screen. 

print("Hello World!")

Output:

Hello World!
>>>

Lesson: 3 - variables

Now lets learn about variables. Strings are basically text. Integers are whole numbers. Floats are numbers that have decimal places. 

myString = "This is a string"
myInteger = 5
myFloat = 3.141592654

print(myString)
print(myInteger)
print(myFloat)

Output:

This is a string
5
3.141592654
>>>

Lesson: 5 - Input

You can use input() to request the user to input some information into the code. Try asking for their name and display it using print().

print(input("Whats your name? "))

Output:

Whats your name? Jordon's Web
Jordon's Web
>>>

Lesson: 6 - Storing Input

You can store the input by setting it to a variable for example: myString = input().

myString = input("What's your name? ")
print(myString)

Output:

What's your name? Jordon's Web
Jordon's Web
>>>

Lesson: 7 - If Statements

You can ask the computer to answer simmple questions. Try asking the computer math problems

if 5+5 == 10:
    print("This is true")

Output:

This is true
>>>

Note that there is double equals in the if statement. This is required as the computer might think that it is a variable. To give a better understanding of what I mean: = means equals but == means ... is equal to ....

Lesson: 8 - Else

You can also make the computer to perform an action if the if statemet is false for example:

if 5+5 == 15:
    print("This is true")
else:
    print("This is False")

Output:

This is False
>>>
